# Módulo de Foco (Pomodoro)

## Visão Geral
Ferramenta de produtividade que auxilia o aluno a manter a concentração durante o estudo usando técnicas como Pomodoro (ciclos de foco e pausa).

## Funcionalidades
- **Timer:** Cronômetro configurável.
- **Estatísticas:** Acompanhamento do tempo de foco.
