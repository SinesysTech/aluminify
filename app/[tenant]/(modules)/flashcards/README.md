# Módulo de Flashcards

## Visão Geral
Ferramenta de revisão espaçada (SRS - Spaced Repetition System) integrada.

## Funcionalidades
- **Revisão (Aluno):** Interface de "Swipe" para estudar cartões e classificar dificuldade.
- **Gestão (Professor/Gestão):** Criação e organização de baralhos e cartões vinculados a módulos/aulas.
