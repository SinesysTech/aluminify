# Módulo Corporativo (Empresa)

## Visão Geral
Gerencia as configurações do Tenant (Cliente SaaS). Define a identidade, regras de negócio e integrações da instituição de ensino.

## Funcionalidades
- **Personalização (White-label):** Configuração de cores, logos e temas (`/personalizacao`).
- **Integrações:** Conexão com ferramentas externas (Zoom, Google, Hotmart).
- **Configurações:** Regras globais da plataforma para o tenant.

## Organização Interna
- `(gestao)`: Painel de administração da empresa.
