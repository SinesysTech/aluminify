# Módulo Dashboard

## Visão Geral
O ponto de entrada do sistema. Atua como um "Hub" que direciona o usuário para as funcionalidades relevantes e apresenta resumos estatísticos.

## Funcionalidades
- **Roteamento Inteligente:** Direciona Alunos, Professores e Admins para suas respectivas visões.
- **Métricas:** Exibe progresso, próximos eventos e alertas.

## Organização Interna
- `(aluno)`: Dashboard do estudante (Gráficos de estudo, Agenda).
- `(institution)`: Dashboard administrativo (Métricas de negócio).
- `(professor)`: Dashboard do docente (Próximas aulas, Alunos sob cuidado).
