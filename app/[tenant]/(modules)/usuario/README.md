# Módulo de Usuários

## Visão Geral
Central de gestão de identidades e perfis. Gerencia alunos, professores e equipe administrativa.

## Funcionalidades
- **Gestão de Alunos:** Matrículas, dados pessoais e histórico.
- **Gestão de Professores:** Perfis e atribuições.
- **Equipe:** Controle de acesso para staff.

## Organização Interna
- `(gestao)`: Tabelas e formulários administrativos divididos por papel (Alunos, Professores, Equipe).
